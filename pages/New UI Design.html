New UI Design

The number pad interface on the iPhone alarm has several pros over the number dial interface, including:

Easy and precise setting of alarm times: The number pad interface allows users to enter the exact time they want to set their alarm, which makes it easier to set precise alarm times.

Quick setting of multiple alarms: The number pad interface allows users to quickly set multiple alarms by entering the desired times for each alarm.
